 Access from Web Browser

 Access from Mobile Devices

There are several options for Mobile apps that works well with Matrix 2.0 federated protocol.
Our recommendations:

  • Element X for Android & iOS

  • Fluffy Chat for Android & iOS

 Access from Desktop Apps

The best option would be Element X client for Desktop, available for most major operating systems

  • Element X for Windows

  • Element X for MacOS

  • Element X for Linux
